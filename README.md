Engima adalah aplikasi web pemesanan tiket bioskop daring.

Berikut adalah fitur yang dimiliki oleh Engima:

* Pembelian tiket
* Pemberian ulasan
* Riwayat pembelian tiket

* MySQL

* Browser dan Database
